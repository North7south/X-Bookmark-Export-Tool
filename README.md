X-Bookmark-Export-Tool

A lightweight, privacy-focused Chrome extension to backup your X (Twitter) bookmarks and profile posts. Built with AI assistance (Cursor + Gemini).

1. Features
- **Auto-Scroll**: Efficiently fetches posts by simulating human scrolling.
- **Year Filter**: Option to export only current year (2026) data.
- **Privacy First**: 100% local execution. Your data never leaves your browser.
- **Clean Export**: Converts raw timestamps into readable local time formats (YYYY-MM-DD HH:MM:SS).

2. Installation (Developer Mode)
- Download this repository as a ZIP and extract it.
- Open Chrome and go to `chrome://extensions/`.
- Toggle on **Developer mode**.
- Click **Load unpacked** and select the extracted folder.

3.How to Use
- Open your X Bookmarks or a Profile page.
- Click the extension icon and select the correct mode.
- Click **Start** to begin scrolling.
- Once finished, click **Export to CSV**.
